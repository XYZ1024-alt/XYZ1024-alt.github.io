使用方法

安装方法

使用的是Ubuntu，所以可以直接使用apt install ffmpeg -y

获取帮助

$ man ffmpeg

FFmpeg的主要语法

$ ffmpeg [global_options] {[input_file_options] -i input_url} ... {[output_file_options] output_url} ...

使用-i来输入文件，例如ffmpeg -i input.avi，这样可以获取文件的细节，例如

查看FFmpeg支持的格式

$ ffmpeg -formats

使用FFmpeg转换格式

$ ffmpeg -i Example.mp4 Example.avi

$ ls
 Example.avi
 Example.mp4

甚至可以转换为.mp3

$ ffmpeg -i Example.mp4 Example.mp3

$ ls
 Example.mp3
 Example.mp4

当然最好使用此命令来移除视频流

$ ffmpeg -i Example.mp4 -vn Example.mp3

也可以移除音频流

$ ffmpeg -i Example.mp4 -an output.mp4

更改媒体的分辨率

$ ffmpeg -i Example.mp4 -s 1280x720 -c:a copy output.mp4

压缩视频

$ ffmpeg -i Example.mp4 -vf scale=1280:-1 -c:v libx264 -preset veryslow -crf 24 output.mp4

压缩音频

$ ffmpeg -i Example.mp3 -ab 128 output.mp3

其中，-ab后面的是比特率
